# WiggleWoo Word Quest 🎉

WiggleWoo Word Quest is an interactive phonics-based learning game designed to help children master foundational reading skills through structured word quests.

Built and maintained by Bottor Technologies Inc.

---

## 🚀 Features

- CVC, CVCC, and vowel-based word quests
- Structured node progression system
- Trophy Room achievement system
- Word-level advancement tracking
- Animated quest map navigation
- Visual word-image matching
- Learning Insights screen for performance feedback

---

## 🧠 Learning Philosophy

WiggleWoo reinforces phonics mastery through:
- Repetition with progression
- Visual association
- Incremental difficulty
- Reward-based engagement

The system is designed to support structured reading development while keeping the experience playful and motivating.

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Local progression storage
- Modular quest architecture

## 📂 Project Structure

- `/screens` – App screens (PlayNow, QuestMap, GameScreen, TrophyRoom)
- `/components` – Reusable UI components
- `/game` – Word data and progression logic
- `/assets` – Images and audio files
